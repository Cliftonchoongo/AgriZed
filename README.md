# AgriZed — Agricultural Market Price Monitor

## Project Overview
A Java backend system built with Javalin that monitors agricultural market prices across Zambia.

## Tech Stack
- Java 25
- Javalin 6
- Gson
- JUnit 5
- Maven

## How to Run
1. Clone the repository
2. Open backend folder in IntelliJ
3. Run Main.java

## OOP Design
- **Encapsulation** — all fields private
- **Inheritance** — Commodity and User are abstract base classes
- **Polymorphism** — formatPrice() differs per commodity type
- **Abstraction** — abstract classes define contracts
